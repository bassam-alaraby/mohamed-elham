# Mohamed & Elham — Wedding Invitation

A simple static wedding invitation page. Tap the wax seal to open the envelope.

## Files

- `index.html` — page structure
- `style.css` — all styling
- `script.js` — envelope animation, countdown, background particles

## How to use

Just open `index.html` in a browser. No build step, no dependencies.

## Customize

- **Names / date / venue**: edit the text inside `index.html`
- **Countdown target**: change the date in `script.js` → `var target = new Date("2026-07-22T19:00:00Z")`
- **Colors**: edit the CSS variables at the top of `style.css` (`:root { ... }`)
